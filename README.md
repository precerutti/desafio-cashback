## Desafio Sistema de Cashback

## Installation and Setup Instructions

`yarn`  

To Run Test Suite:  

`yarn test`  

To Start Server:

`yarn start`  

## Stack

  - ReactJs (Create React App)
  - Typescript
  - ChakraUI
  - Jest/Testing Library
  - MirageJs
  - Axios
  - Styled Components
  - ContextAPI
